# Structure des assets

- /sprites : images, tilesets, animations
- /sons : effets sonores (mp3, wav)
- /musique : musiques de fond
- /cartes : données de niveaux ou maps (json, tmx, etc.)
- /autres : tout autre type de ressource

## Conseils

- Utilise des noms explicites et cohérents
- Privilégie les formats standards (png, mp3, json)
- Range chaque ressource dans le bon dossier
- Mets à jour ce README si tu ajoutes de nouveaux types d’assets
